\begin{question}{Create a Table / Tabelle Erstellen}%
\begin{itemize}%
\item[EN] Provide the \sqlDialect\ command to create a table \inQuotes{cars} with the following columns:%
\begin{itemize}%
\item \inQuotes{name}: a character string of reasonable length~(must be unique),%
\item \inQuotes{price}: the costs of one such car in RMB~(must always be provided),%
\item \inQuotes{top\_speed}: the maximum speed.
\end{itemize}%
%
\item[DE] Geben Sie das \sqlDialect\ Kommand an, um eine Tabelle \inQuotes{cars} mit den folgenden Spalten zu erstellen:%
\begin{itemize}%
\item \inQuotes{name}: ein Text vernünftiger Länge, der eindeutig/einmalig sein muss,%
\item \inQuotes{price}: die Kosten eines solchen Autos in RMB~(muss immer angegeben werden),%
\item \inQuotes{top\_speed}: die Maximalgeschwindigkeit.
\end{itemize}%
\end{itemize}%
\end{question}%
