\item name: the full name of the person,%
\item dob: the date of birth of the person,%
\item province: the province where the person lives,%
\item city: the city where the person lives,%
\item address: the street address of the person.%
\end{itemize}%
}{%
Nehmen Sie an, dass eine Tabelle Daten über Leute in den Spalten wie unten angegeben speichert. %
Erklären Sie, welche Spalten wahrscheinlich funktional von welchen anderen Spalten abhängen werden.%
\begin{itemize}%
\item national\_id: die chinesische Ausweisnummer,%
\item mobile: die Mobiltelefonnummer,%
\item name: der volle Name der Person,%
\item dob: das Geburtsdatum der Person,%
\item province: die Provinz, in der die Person lebt,%
\item city: die Stadt, in der die Person lebt,%
\item address: die Adresse (Straße, Hausnummer, \dots), unter der die Person lebt.%

}%

\begin{questionENDE}{First Normal Form}{Erste Normalform}%
\qENDE{%
The table \emph{guest} has the columns guest\_id, guest\_family\_name, guest\_given\_name, guest\_address\_1, guest\_address\_2, guest\_mobile\_1, guest\_mobile\_2. %
It is used to store the information of guests of a hotel.%
\begin{enumerate}%
\item Find the proper key for the table.%
\item What happens if a guest wants to provide three contact addresses?%
\item Transform the table to the first normal form~(\glsShort{1NF}).%
\end{enumerate}%
}{%
Die Tabelle \emph{gast} hat die Spalten gast\_id, gast\_familienname, guest\_vorname, gast\_adresse\_1, gast\_adresse\_2, gast\_mobil\_1, gast\_mobil\_2. %
Sie wird verwendet, um die Informationen über Gäste eines Hotels zu speichern.%
\begin{enumerate}%
\item Finden Sie die passenden Schlüssel der Tabelle.%
\item Was passiert, wenn ein Gast drei Kontaktadressen hinterlegen möchte?%
\item Überführen Sie die Tabelle in die erste Normalform~(\glsShort{1NF}).%
\end{enumerate}%
}%
\end{questionENDE}%
